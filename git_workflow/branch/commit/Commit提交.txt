> git add -u 添加所有的updates files到暂存区(其中的改变将会在下一次的提交中添加)
> git add -A 添加所有的文件，包括没有追踪的文件
> git add .  添加当前分支所有被追踪的文件

> git commit -am "..." 将add和commit合为一步
